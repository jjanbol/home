PEFT Analysis with Custom LoRA-Enhanced GPT-2 on the Rotten Tomatoes Dataset

Fine-tuned GPT2 model using custom written Low-Rank Adaptation (LoRA) to achieve parameter-efficient learning PEFT for sentiment analysis on the Rotten Tomatoes dataset. The work involves modifying the transformer architecture by integrating trainable low-rank matrices into the attention layers affecting the query, key, and value matrices, reducing the number of trainable parameters to approximately 5.3% of the original model while preserving inference speed. Experiments were conducted, comparing various hyperparameter configurations (different LoRA ranks and scaling factors) against full fine-tuning, with performance metrics evaluated through test accuracy and validation loss curves. Please request the repository access through email or linkedin if interested.

Implementation of Time-Reversed Language Models (TRLMs) to improve citations for documents that require precision

This project explores whether Time-Reversed Language Models (TRLMs) can improve citation accuracy by addressing the "hallucination" problem in language models. The work, building on research by Varun et al. (2025), aims to test the effectiveness of reverse language models in citation performance. Initial fine-tuning tests have been conducted, with plans for further optimization and an ablation study to improve model performance. Once completed, citation accuracy is expected to improve noticeably on documents that are used in critical industries such as law, healthcare, engineering, etc.
